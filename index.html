Welcome to QuickBox project.

This project is mainly QtDeclarative platform for QWidgets. This is different approach that Digia has. They are creatings QWidgets using Quick2, it is so called QuickControls. I'm bringing QWidgets to the declarative environment, so one can use old good QWidgets in QML. To expose QWidget based controls to QML, they have to be lightly overridden and registered to declarative framework in QML plugin. There are two such a plugins currently implemented:

  • qfcoreplugin - core QML objects (SQL, cryptography, networking, etc. )
  • qfqmlwidgetsplugin - GUI stuff, QWidget based controls, framework classes (MainWindow, PluginLoader, etc. )
